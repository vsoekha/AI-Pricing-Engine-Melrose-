# Customer Onboarding Process

## Stap-voor-stap proces om klanten aan boord te krijgen

### Fase 1: Sales & Demo (Week 1)

1. **Initial Contact**
   - Demo aanvraag
   - Discovery call (wat hebben ze nodig?)

2. **Demo Presentatie**
   - Live demo van het systeem
   - Use cases relevant voor hun business
   - Q&A sessie

3. **Proposal**
   - Pricing voorstel
   - Timeline
   - Deliverables

### Fase 2: Data Collection (Week 2-3)

1. **Data Template Versturen**
   - Excel/CSV template
   - Data dictionary (wat betekent elke kolom)
   - Voorbeelden

2. **Data Gathering Support**
   - Help klant data verzamelen
   - Data quality checks
   - Missing data identificatie

3. **Data Validation**
   - Automatische validatie
   - Quality report
   - Go/no-go beslissing

### Fase 3: Custom Model Training (Week 3-4)

1. **Data Import**
   - Import in database
   - Data cleaning
   - Feature engineering

2. **Model Training**
   - Train custom model
   - Hyperparameter tuning
   - Cross-validation

3. **Model Evaluation**
   - Performance metrics
   - Comparison met baseline
   - Validation report

### Fase 4: Deployment (Week 4-5)

1. **API Setup**
   - Custom API endpoint
   - Authentication setup
   - Rate limiting

2. **Frontend Integration**
   - White-label frontend
   - Custom branding
   - User training

3. **Go-Live**
   - Production deployment
   - Monitoring setup
   - Support handoff

### Fase 5: Support & Optimization (Ongoing)

1. **Monitoring**
   - API usage tracking
   - Model performance monitoring
   - Error tracking

2. **Model Updates**
   - Periodic retraining (bijv. elk kwartaal)
   - New data integration
   - Performance improvements

3. **Support**
   - Helpdesk
   - Feature requests
   - Training sessions

## Data Collection Template

### Excel Template voor Klanten

**Kolommen:**
- Property ID (uniek identifier)
- Asset Type (logistics/office/resi)
- City
- Size (m²)
- Quality Score (0-1)
- NOI Annual (€)
- Cap Rate Market (decimaal, bijv. 0.065)
- Interest Rate (decimaal, bijv. 0.025)
- Liquidity Index (0-1)
- List Price (€)
- Comp Median Price (€)
- **Sold Within 180d** (0 of 1) - **BELANGRIJK**
- Sale Date (als verkocht)
- Sale Price (als verkocht)

### Data Requirements

**Minimum:**
- 50 properties met bekende verkoopstatus
- Alle verplichte kolommen ingevuld
- Realistische waarden

**Idealiter:**
- 200+ properties
- Diverse portfolio (verschillende types, steden)
- Recente data (laatste 2-3 jaar)
