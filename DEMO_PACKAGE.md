## 2. Database Structuur voor Klantdata

Je moet een database structuur hebben om klantdata op te slaan en te trainen.

### Opties:

**Optie A: PostgreSQL Database (Aanbevolen)**
- Professioneel
- Schaalbaar
- Goed voor productie

**Optie B: SQLite (Voor demo/small scale)**
- Eenvoudig
- Geen server nodig
- Goed voor MVP

**Optie C: Cloud Database (Render PostgreSQL)**
- Managed service
- Automatische backups
- Schaalbaar
